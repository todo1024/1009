课程介绍：

什么是Lightroom？

1、Lightroom是当今数字拍摄工作流程中不可或缺的一部分。

2、拥有增强的校正工具、强大的组织功能以及灵活的打印选项。

3、是适合设计师、摄影师输入、选择、修改和大批量展示数字图像的高效率软件。

为什么选择用Lightroom进行后期修图？

1、批量处理：Lightroom一次可以处理大量照片，更加高效快捷。

2、直观简洁：Lightroom的页面直观简洁，拥有简单的调色功能、强大的预设功能，能够满足修图需求。
